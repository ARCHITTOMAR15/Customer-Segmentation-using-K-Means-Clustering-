# Customer-Segmentation-using-K-Means-Clustering-
Customer segmentation project using unsupervised machine learning to identify distinct customer groups based on demographic and spending behavior. Includes EDA, data preprocessing, outlier detection, and K-Means clustering to derive actionable business insights.

**Project Overview**

Customer segmentation is a crucial technique in marketing analytics that helps businesses understand their customers and target them effectively.

This project applies unsupervised machine learning techniques to segment customers based on their demographic and spending behavior. By identifying patterns in customer data, businesses can design targeted marketing strategies, improve customer retention, and maximize revenue.

The project demonstrates the complete machine learning workflow, including:

Data exploration

Data preprocessing

Outlier detection

Feature analysis

Clustering using K-Means

Customer segment interpretation

**Project Highlights**

Applied Unsupervised Machine Learning for customer segmentation  

Implemented K-Means clustering with optimal cluster selection using the Elbow Method  

Performed Exploratory Data Analysis and feature analysis 

Identified distinct customer groups based on income and spending behavior  

Generated actionable marketing insights

**Dataset**

The dataset used in this project contains customer information from a retail store:
Features

Feature	Description

Gender	Gender of the customer

Age	Customer age

Annual Income (k$)	Annual income in thousand dollars

Spending Score (1–100)	Score assigned based on customer spending behavior

Customer segmentation revealed different behavioral groups such as:
High income – high spending customers

High income – low spending customers

Low income – high spending customers

Average income – moderate spending customers
