# Customer-Segmentation-using-K-Means-Clustering-
Customer segmentation project using unsupervised machine learning to identify distinct customer groups based on demographic and spending behavior. Includes EDA, data preprocessing, outlier detection, and K-Means clustering to derive actionable business insights.

**Project Overview**

Customer segmentation is a crucial technique in marketing analytics that helps businesses understand their customers and target them effectively.

This project applies unsupervised machine learning techniques to segment customers based on their demographic and spending behavior. By identifying patterns in customer data, businesses can design targeted marketing strategies, improve customer retention, and maximize revenue.

The project demonstrates the complete machine learning workflow, including:

1.Data exploration

2.Data preprocessing

3.Outlier detection

4.Feature analysis

5.Clustering using K-Means

6.Customer segment interpretation

**Project Highlights**

a.)Applied Unsupervised Machine Learning for customer segmentation  

b.)Implemented K-Means clustering with optimal cluster selection using the Elbow Method  

c.)Performed Exploratory Data Analysis and feature analysis 

d.)Identified distinct customer groups based on income and spending behavior  

e.)Generated actionable marketing insights

**Dataset**

The dataset used in this project contains customer information from a retail store:
Features

1.Feature	Description

2.Gender	Gender of the customer

3.Age	Customer age

4.Annual Income (k$)	Annual income in thousand dollars

5.Spending Score (1–100)	Score assigned based on customer spending behavior
